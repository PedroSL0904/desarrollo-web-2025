Amazon Web Services

  2. Ventajas
  3. Servicios representativos

Definición

Amazon Web Services (AWS) es una plataforma de servicios en la nube que ofrece infraestructura y servicios bajo demanda para computación, almacenamiento, bases de datos, redes, análisis, inteligencia artificial y muchas otras capacidades.

Características principales

AWS proporciona una gran variedad de servicios listos para usar que permiten a organizaciones evitar la inversión en hardware físico y pagar según el uso real de recursos.

La plataforma está organizada en regiones geográficas, cada una con varias Zonas de Disponibilidad (AZ) físicas diseñadas para aislar fallos y ofrecer alta disponibilidad.

Servicios representativos

Amazon EC2: servicio de computación que permite lanzar máquinas virtuales (instancias) con distintos tamaños y sistemas operativos.

Amazon S3: servicio de almacenamiento de objetos, escalable y duradero, utilizado para copias de seguridad, hosteo de sitios estáticos y almacenamiento de datos para análisis.

Amazon RDS: servicio administrado de bases de datos relacionales compatible con motores como MySQL, PostgreSQL y SQL Server.

AWS Lambda: plataforma serverless que ejecuta código en respuesta a eventos sin administrar servidores.

DynamoDB: base de datos NoSQL gestionada de alta disponibilidad y baja latencia.

Ventajas

Escalabilidad: los recursos se pueden ajustar rápidamente según la demanda, permitiendo responder a picos de tráfico sin aprovisionamiento físico previo.

Modelo de pago por uso: se factura según el consumo real de recursos, lo que reduce costes iniciales y convierte la inversión en gasto operativo.

Amplio catálogo de servicios: AWS ofrece cientos de servicios que cubren desde infraestructuras básicas hasta capacidades avanzadas de datos e inteligencia artificial.

Casos de uso comunes

Hospedaje y despliegue de aplicaciones web y móviles, almacenamiento y copia de seguridad de datos, análisis de grandes volúmenes de información, entrenamiento y despliegue de modelos de machine learning, y entornos de desarrollo y pruebas (DevOps).

Seguridad y cumplimiento

AWS incluye controles y servicios para gestión de identidades, cifrado, registro de auditoría y protección frente a ataques, además de cumplir múltiples estándares de seguridad y normativas internacionales.

Cómo empezar

Para comenzar con AWS se crea una cuenta, se elige la región adecuada, y se puede iniciar con servicios básicos como EC2 para computación y S3 para almacenamiento; AWS ofrece documentación y guías para cada servicio.

Resumen

AWS es una plataforma en la nube amplia y flexible que permite a organizaciones de todos los tamaños construir, escalar y operar aplicaciones sin gestionar infraestructura física, con un modelo de pago por uso y presencia global mediante regiones y zonas de disponibilidad.
